# XYZSOlve — Digital Solutions Agency Website 

## 1. Project Description

XYZSOlve is a fully responsive digital agency website developed as a college academic project.  
The website simulates a real-world software company that offers professional digital services such as web development, mobile applications, UI/UX design, and enterprise solutions.

The project demonstrates a complete client workflow:
Service Selection → Package Customization → Order Review → Checkout → Confirmation.

This project focuses on frontend development, responsive design, and dynamic UI behavior using JavaScript.

---

## 2. Project Objectives

- Design a professional multi-page business website
- Implement responsive layouts for all devices
- Create a dynamic service and package selection system
- Simulate a real agency ordering workflow
- Apply modern UI/UX principles

---

## 3. Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript (DOM Manipulation, localStorage)
- Font Awesome Icons

No backend or database integration is used in this version.

---

## 4. Website Pages

index.html  
→ Homepage with services, portfolio, contact section, and footer.

about.html  
→ Company introduction and team members section.

request.html  
→ Service request form where users select a service.

review.html  
→ Package selection page with dynamic add-ons and price calculation.

checkout.html  
→ Payment-style checkout page.

confirmation.html  
→ Order confirmation page.

contact.html  
→ Dedicated contact page for customer queries.

---

## 5. Key Features

### Responsive Design
- Fully responsive across desktop, tablet, and mobile devices
- Media queries implemented for layout adjustment

### Dynamic Service Flow
- Selected service stored using localStorage
- Review page auto-updates based on chosen service
- Professional add-ons change according to selected service

### Package System
- Basic, Standard, and Premium packages
- Real-time price calculation
- Add-on services with automatic total update

### Professional UI Components
- Sticky navigation bar
- Modern service cards
- Portfolio grid layout
- Team section with professional icons
- Structured footer with contact information
